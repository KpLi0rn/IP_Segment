# IP_Segment

企业IP网段信息搜集工具V1.1

### 使用说明
环境依赖安装
`pip install -r requirements.txt`
复制需要进行网段搜集的域名信息
然后运行
`python IP_Segment.py`
就可以得到结果，结果在result文件夹中
